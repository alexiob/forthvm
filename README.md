# ForthVM

Very simple Forth VM/Interpreter written in Elixir

- [x] tokenizer
- [x] interpreter
- [x] core words
- [x] loops
- [x] if-then
- [x] include
- [ ] REPL
- [ ] sleep word
- [ ] VM
- [ ] multiple Forth cores running in a single VM
- [ ] process messages: sending messages to a process will call a matching word
- [ ] step-by-step debugger
